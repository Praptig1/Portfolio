# Prapti Gurung — Portfolio

This is my personal portfolio website, built to showcase my projects, experience, and interests in software development, creative technology, VR, and human-centered design.

## About

I’m a Computer Science and Psychology graduate from the University of Victoria. My work explores the intersection of software, design, psychology, and creative technology, with projects across web development, VR, and user-centered digital experiences.

## Featured Projects

### VR Emotion Regulation Application

A Unity VR application designed to help students manage exam anxiety through guided breathing, grounding, real-time check-ins, and reflective feedback.

Tech: Unity, C#, XR Interaction Toolkit, Unity EventSystem

### Nest — Nature Journaling App

A full-stack nature journaling app that lets users upload, organize, and revisit geotagged nature observations through a calming interface.

Tech: Next.js, React, TypeScript, Supabase, Apollo Client, Lottie

### Full Recipe App

A dynamic recipe search app that uses the Spoonacular API to help users browse recipes by cuisine, diet, and cooking time.

Tech: React, JavaScript, CSS, HTML, Spoonacular API
